また、この作品ではJavaScriptでフォームに正しく入力されているか確認するシステムを構築しました。
すべての必須項目に入力がされていないと先のページに進めないようになっています。

CSSの長さが非常に長くなってしまったため、途中で横着状態になってしまったことは反省点です。
